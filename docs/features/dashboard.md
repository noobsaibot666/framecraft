# Dashboard (`/`)

Production workspace home — library totals, activity health, and quick jumps back into recent work.

## What you can do
- Search across recent prompts, top rated, and recent results at once (filters everything below live)
- See first-run onboarding when the library is empty, with a 3-step guide (Craft → Queue/Generate → Import/Review)
- Jump straight into Prompt Craft, a new Project, the Queue, Import, Recipes, or SREFs
- Resume the last prompt you touched in one click
- Drill into any stat, tag, provider, project, or token to jump to its filtered view

## Header actions
New Project · Prompt Craft

## Page sections — top to bottom
1. **Search bar** — filters recent prompts, top rated, and recent results by title/text/tag/notes.
2. **First-run guide** — shown only when the library has zero prompts; disappears after the first prompt is created.
3. **Library totals** — Total Prompts, Total Results, Winners, Win Rate, No Results (prompts never generated).
4. **Recent Wins** — last 4 winning results as thumbnail cards with score; link to full winners view.
5. **Top Tags** — most-used tags as click-through chips into the filtered library.
6. **Provider Performance** — win rate per provider with prompt/win counts; click through to that provider's library view.
7. **Production Health (this week)** — Prompts Crafted, Results Reviewed, Win Rate, Needs Review (alert state), Active Projects, Queue Depth (alert state), plus a 7-day prompts/results activity sparkline.
8. **Continue where you left off** — one-click resume of the most recently touched prompt.
9. **Main grid**:
   - **Recent Prompts** (left, collapsible) — latest library entries, click through to detail.
   - **Recent Results** — latest imported/generated results with score dots.
   - **Active Projects** — open projects with prompt counts.
   - **Needs Review** — results awaiting a rating.
   - **Proven Tokens** — highest quality-score tokens from usage.
   - **Winner Tokens** — tokens most correlated with winning results, ranked.
   - **Top Rated** — highest-scored prompts in the library.
10. **Quick Actions** — Queue (with live depth badge), Import, Recipes, SREFs.
